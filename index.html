CSCloud

CSCloud (cscloud.lnu.se) is our self hosted Open Stack environment in which we can create virtual machines. (Infrastructure as a Service, IaaS)

In your "Secrets" project on Gitlab you will find crendentials for logging in to cscloud.lnu.se. To be able to connect to CSCloud you need to use a VPN.

VPN (Virtual Private Network)

To protect the machines from attacks, CSCloud is hidden behind a firewall. To be able to access CSCloud from outside of the university network you will need to connect through a VPN. Make sure you connect through the VPN when trying to connect to CSCloud.

Guide for connecting to the VPN

Getting started

To be able to create machine in an easy way we need to be able to work with cscloud through its API using a CLI (Command line interface).

Use the "Introduction to CSCLOUD" guide by Morgan Ericsson to start to get familier with the environment.
